# 🚍 BusHub Zambia

BusHub Zambia is a complete bus booking platform designed for the Zambian market.

## Features

- 🔍 Passenger Booking - Search buses, select seats, book tickets
- 👑 Super Admin Dashboard - Manage companies, pending approvals
- 🏢 Company Admin Dashboard - Manage branches, view bookings
- 🏪 Branch Manager Dashboard - Add routes, sell tickets
- 🔐 Role-Based Access - Different dashboards for different roles
- 📱 Mobile Responsive - Works on all devices
